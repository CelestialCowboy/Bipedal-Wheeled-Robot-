A custom bipedal wheeled robot. Runs in CPP based on the simpleFOC library.
**Critical Hardware:**

| Item                          | Quantity |
|-------------------------------|----------|
| ESP32-Wroom                   | 1        |
| DRV8302                       | 2        |
| MPU6050                       | 1        |
| DC-DC StepUP (18 → 24)        | 1        |
| DC-DC StepDOWN (18 → 6)       | 1        |
| 360kV BLDC Motor              | 2        |
| PCA9548A                      | 1        |
| AS5600 Hall Effect Sensor     | 2        |
| 18V 2Ah Dewalt Battery        | 1        |
| Dewalt Battery Adapter        | 1        |
