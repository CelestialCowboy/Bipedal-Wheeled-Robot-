A custom bipedal wheeled robot. 
Runs in CPP based on the simpleFOC library.
Hardware:
- ESP32-Wroom x1
- DRV8302 x2
- MPU6050 x1
- DC-DC StepUP x1
- DC-DC StepDOWN x1
- 360kV BLDC motor x2
- PCA9548A x1
- 18V 2Ah Dewalt Battery x1
- AS5600 Hall Effect Sensor x2
